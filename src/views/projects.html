Promathius Web Game

A turn-based strategy game built in PHP and MySQL, leveraging a standard HTML/CSS/Javascript frontend (with a few experiments using Cordova for the mobile app).


Signicant enhancements since it's predecessor. Features separation between backend and front-end code using the Smarty Template Engine. Significant portions of the game engine were rewritten to be more modular. Game configurations are separated into user-friendly .ini configs. Also features significant performance enhancements, automatically compressing and minifing JS and CSS, as well as caching re-usable elements.

Angular Commerce

A dynamic ecommerce front-end framework using AngularJS and the Moltin API. Supports standard ecommerce functionality such as categories, product lists, cart panel, checkout and payment.

Shopify Variant Plugin

Shopify currently lacks the option to feature multiple variant-specific images. This plugin addresses that issue by providing an easy to implement JS snippet that can filter images based on the Shopify alt tag.
